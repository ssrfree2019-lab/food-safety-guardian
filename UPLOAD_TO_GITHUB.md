# 上传到 GitHub

1. 打开仓库：https://github.com/ssrfree2019-lab/food-safety-guardian
2. 点击 **Add file** → **Upload files**。
3. 将本文件所在文件夹内的全部文件和文件夹拖入上传区域。
4. 不要额外上传 `node_modules`、视频 MP4、缓存或渲染目录。
5. 提交说明填写：`Add food safety guardian video source project`。
6. 点击 **Commit changes**。

上传后仓库将包含：

- HyperFrames HTML/GSAP 合成源码
- Remotion React/TypeScript 源码
- 中文字幕与旁白脚本
- 视觉规范与验证说明

最终 MP4 成片继续作为独立交付保存；以后如需在 GitHub 分发视频，建议使用 Releases 或 Git LFS。
